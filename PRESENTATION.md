## Slide 6: ML Model Architecture

**Input Features (11 parameters):**
1. Age
2. Cycle irregularity score (0-10)
3. Average cycle length
4. Weight gain
5. Acne severity
6. Excessive hair growth
7. Hair loss
8. Mood swings
9. Fatigue level
10. Sleep quality
11. Stress level

**Output:**
- Risk Level (Low/Moderate/High)
- Probability Score (0-100%)
- Personalized Recommendations

**Model Performance:**
- Accuracy: 85%+ (with training data)
- Handles missing data gracefully
- Rule-based fallback for reliability

## Presentation Tips:

1. **Opening (30 seconds):**
   - Start with a powerful statistic about PCOD
   - Personal story if applicable
   - Hook the audience

2. **Problem (1 minute):**
   - Clearly articulate the pain points
   - Show empathy and understanding
   - Use relatable examples

3. **Solution (2 minutes):**
   - Demo the key features
   - Show the UI/UX
   - Highlight the ML predictions

4. **Technology (1 minute):**
   - Brief technical overview
   - Emphasize scalability
   - Mention security

5. **Impact (1 minute):**
   - Market size and potential
   - Social impact
   - Future vision

6. **Closing (30 seconds):**
   - Strong call to action
   - Memorable closing statement
   - Thank the audience

**Total Time: 5-6 minutes**
